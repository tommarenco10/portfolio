Gestión de clubes

En la universidad junto a un equipo desarrollé una aplicación en ASP.NET Web Forms con SQL Server para la gestión integral de un club deportivo. Tecnologias aplicadas:

  • WebForms
  • SQL server
  • Bootstrap

Gestión de concesionaria

Diseñé una aplicación de gestión de concesionarias de automóviles para un cliente, permitiendo el control de stock de vehículos, clientes, ventas y reportes. El sistema facilita la administración de operaciones comerciales de punta a punta, optimizando la gestión de inventario y el proceso de ventas.

Tecnologias aplicadas:

  • Winforms
  • SQL server

Sistema de gestión bancaria

En conjunto con mi equipo de la universidad implementé un sistema en Java con Tomcat y MySQL para la gestión de un banco. Desde el lado del administrador, el sistema ofrece herramientas para aprobar o rechazar préstamos, administrar cuentas y gestionar usuarios.

Tecnologias aplicadas:
